Pasos para el hello world
1) Crear un venv

2) Activar el venv

3) Instalar Django con pip
python -m pip install Django

4) Forma alternativa
   1) instalar venv
   2) activar venv
   3) crear archivo requirements.txt y agregar dependencia Django
   4) pip install -r requirements.txt
   5) pinear versiones con 
      pip freeze -r requirements.txt y reemplazar los valores en el archivo
